# 📇 名片通訊錄 (Business Card Scanner)

一個免費、離線優先的 PWA 應用，用於掃描名片、管理聯絡人，並追蹤業務跟進狀態。

## ✨ 功能

- **📷 智能名片掃描** - 使用 Tesseract.js OCR 技術，完全免費
- **📋 聯絡人管理** - 新增、編輯、刪除、搜尋聯絡人
- **📊 統計分析** - 跟進狀態分佈、來源場合分析
- **⏰ 跟進提醒** - 設置日期提醒，不漏掉重要聯絡人
- **📥 CSV 匯出/匯入** - 輕鬆備份和轉移資料
- **🔄 完全離線** - Service Worker 支援，無網路也能使用
- **📱 PWA 安裝** - 一鍵安裝到手機主畫面

## 🚀 快速開始

1. 訪問應用網址
2. 用手機瀏覽器打開
3. 點擊瀏覽器右上角選單 → 「加到主畫面」
4. 享受離線使用！

## 📖 使用說明

### 掃描名片
1. 點擊「掃描」頁籤
2. 拍照或上傳名片圖片
3. 點擊「開始辨識」
4. 手動檢查和編輯辨識結果
5. 自動保存到聯絡人清單

### 管理聯絡人
- **新增**: 點擊「手動新增」或掃描名片
- **編輯**: 點擊聯絡人卡片 → 編輯
- **刪除**: 在詳情頁點擊刪除按鈕
- **搜尋**: 使用搜尋欄查詢

### 跟進追蹤
1. 打開聯絡人詳情
2. 更改「跟進狀態」
3. 添加「跟進記錄」
4. 設置「提醒日期」

### 備份資料
- **匯出**: 點擊「匯出 CSV」下載資料
- **匯入**: 點擊「匯入」上傳 CSV 檔案

## ⚙️ 跟進狀態

- **待聯繫** (new): 新增的聯絡人
- **已聯繫** (contacted): 已主動聯繫
- **報價中** (quoting): 進行中的報價
- **成交** (closed): 已成交
- **冷淡** (cold): 暫無進展

## 💾 資料存儲

所有資料存儲在瀏覽器的 localStorage 中，完全不上傳到伺服器。請定期匯出備份。
